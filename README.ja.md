# godot-D_Lifesaver

[English](README.md) | [日本語](README.ja.md)

ツールバーに「Life Saver」ボタンを追加し、Git へのステージングと仮コミットをワンクリックで実行できる Godot エディタプラグイン。
作業の区切りで手軽にバックアップを作成し、予期せぬトラブルから作業内容を守る。

![Life Saver Button](doc_images/life_saver_button.jpg)

## 主な機能

- **ワンクリック・バックアップ**: すべての変更をステージング (`git add -A`) し、タイムスタンプ付きのメッセージ (`d_lifesaver: auto-save ...`) で仮コミットを作成する。
- **ショートカット実行**: `Ctrl + Alt + S` (カスタマイズ可能) で、マウスを使わずに即座にバックアップを実行できる。
- **メモ付き保存**: ボタンを **右クリック** することで、任意のメモを追加したコミットを作成可能。
- **一括整理 (Squash)**: 溜まった仮コミットを一つの「Refined」コミットに統合し、履歴をクリーンに保つ。
- **直前の取り消し (Undo)**: 直前の仮コミットを `git reset --soft` で取り消し、ワークツリーの変更を保持したまま以前の状態に戻す。
- **フォーカス喪失時のオートセーブ**: エディタから別のウィンドウに切り替えた際、自動的にバックアップを作成する設定が可能。
- **視覚的なステータス表示**:
  - **Safe (緑色)**: 保存する変更がない状態を自動検知し、`(Safe)` と表示する。
  - **警告 (オレンジ)**: 未保存の変更がある場合にボタンがオレンジ色に変化する。
  - **緊急 (赤色)**: 最後の保存から10分以上経過すると文字が赤くなり、バックアップを促す。
- **詳細な情報表示**:
  - **経過時間**: 最後にコミットしてからの経過時間を表示（例: `15m`）。
  - **変更ファイル数**: 現在の未コミット変更数を表示。
  - **コミットカウンター**: 現在のブランチにある仮コミットの総数を表示（例: `x3`）。
- **履歴とブランチの確認**:
  - ツールチップで現在作業中のブランチ名と、**直近3件のセーブ履歴**を確認できる。
  - ボタン上に現在のブランチ名が常に表示される。
- **トースト通知**: 操作の結果やエラーをエディタ上にポップアップ通知する。
- **カスタマイズ可能**: 各種チェック間隔やショートカットキーは **エディタ設定** から変更可能。

## インストール方法

1. `addons/godot-d_lifesaver` ディレクトリをプロジェクトの `res://addons/` フォルダにコピーする。
2. **プロジェクト > プロジェクト設定 > プラグイン** から `godot-D_Lifesaver` を有効にする。

## 設定

**エディタ > エディタ設定 > d_lifesaver** から以下の設定が可能：
- **Intervals (間隔)**:
  - `Dirty Check Seconds`: 変更の有無を確認する間隔（デフォルト: 2秒）。
  - `Commit Count Seconds`: 仮コミット数を再取得する間隔（デフォルト: 60秒）。
- **Auto Save (オートセーブ)**:
  - `On Focus Loss`: エディタのフォーカスが外れた際に自動保存するか（デフォルト: false）。
- **Shortcut (ショートカット)**:
  - `Trigger`: バックアップを実行するキー組み合わせ（デフォルト: `Ctrl+Alt+S`）。
